# Installation

official installation

In this version, I will only cover UEFI's installation

# Download Archlinux iso file

Archlinux iso

# Make USB Startup Disk

  1. find you usd device location

    df -h
    /dev/sdb? (/dev/sdb1 /dev/sdb2 ...)

  2. umount your usb device (or maybe you can ignore)

    umount /dev/sdb?

  3. make

    sudo dd bs=4M if=/iso-location of=/usb-dev-location

# Start

  1. change start config

    make your computer start from usb

  2. check your boot mode

    UEFI or BIOS ?
    check if /sys/firmware/efi existss
    #!/bin/bash
    [ -d /sys/firmware/efi ] && echo UEFI || echo BIOS

  3. connect to network
    • Mac address

      cat /sys/class/net/enp1s0/address

    • wifi

      wifi-menu
      then choose your available wifi to connect

    • ethernet

      dhcpcd

    • check your connection

      ping www.baidu.com

  4. synchronize your time

    timedatectl set-ntp true

  5. disk partition (UEFI + GPT)
    • view your partition information

      fdisk -l
      you can find your disk with /dev/sda?

    • start partition
      fdisk /dev/sda?
      • fdisk opertation
        • m help comman
        • g to construct gpt
        • n to add partition
        • w write to disk
      • enter g to create a empty GTP partition
      • enter n to create new partition
      • first partition (startup disk)

        /dev/sda1 +512M :: need change the steps.

      • second partition (/boot)

        /dev/sda2 +600M

      • third partition (/)

        /dev/sda3 +40G

      • forth partition (swap)

        /dev/sda4 +1G

      • fifth partition (/home)

        /dev/sda5 +(all left)

    • check your partition

      lsblk

  6. format your disk partition

    mkfs.fat -F32 /dev/sda1
    mkfs.ext4 /dev/sda2
    mkfs.ext4 /dev/sda3
    mkfs.ext4 /dev/sda4
    mkfs.ext4 /dev/sda5

  7. mount disk
    • swap
      mkswap /dev/sda4
      swapon /dev/sda4
    • mount
      mount /dev/sda3 /mnt
      mkdir /mnt/boot
      mount /dev/sda2 /mnt/boot
      mkdir /mnt/home
      mount /dev/sda5 /mnt/home
    • check your mount
      lsblk
  8. change your archlinux mirros
    nano /etc/pacman.d/mirrowlist

    Server = http://mirrors.tuna.tsinghua.edu.cn/archlinux/$repo/os/$arch
    Server = http://mirrors.zju.edu.cn/archlinux/$repo/os/$arch
    Server = https://mirrors.163.com/archlinux/repo/os/arch

  9. install basic packages

    pacstrap -i /mnt base base-devel

  10. fstab config
    • generate fstab file

      genfstab -U /mnt >> /mnt/etc/fstab

    • check /mnt/etc/fstab

      cat /mnt/etc/fstab

  11. Chroot

    arch-chroot /mnt /bin/bash

  12. Setup Time ZOne

    ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime hwclock --systohc

  13. Install Applications

    pacman -S vim dialog wpa_supplicant ntfs-3g networkmanager

  14. Localization

    nano /etc/locale.gen
    uncomment en_US.UTF-8 zh_CN.UTF-8
    locale-gen
    echo LANG=en_US.UTF-8 > /etc/locale.conf

  15. root password

    passwd

  16. add user

    useradd -m -g users -s /bin/bash xxx(username)
    passwd xxx
    nano /etc/sudoers add xxx ALL=(ALL) ALL

  17. Intel-ucode (only for Intel CPU)

    pacman -S intel-ucode

  18. boot loader

    pacman -S grub efibootmgr
    grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=grub --recheck

    • you can check your /boot

      cd /boot
      ls

    grub-mkconfig -o /boot/grub/grub.cfg

  19. reboot

    exit
    reboot
